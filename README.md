# PhantomCipher v1.0 🔐
**A Professional Command-Line Encryption Suite for Security Enthusiasts.**

PhantomCipher is a lightweight yet powerful CUI-based encryption tool developed in Python. It offers multiple layers of security, including Caesar Cipher, XOR logic, Base64 encoding, and SHA-256 hashing. Designed specifically for terminal lovers and mobile-based environments like Pydroid3.

---

## ✨ Key Features
- 🛠 **Multi-Layer Encryption:** Combines Caesar, Base64, and XOR for maximum security.
- 🧪 **Hash Generator:** Supports MD5, SHA1, SHA256, and SHA512.
- ⚡ **Interactive CUI:** Smooth terminal experience with loading animations and color codes.
- 🔓 **Brute-Force Simulator:** Simulate a Caesar Cipher brute-force attack.
- 📱 **Mobile Ready:** Fully compatible with Pydroid 3 (Android) and Linux Terminals.
